# 🌿 Plant Disease Prediction AI

## 📌 Problem Statement
Crop diseases are a major threat to food security, but rapid identification remains difficult in many parts of the world. An automated, cost-effective, and highly accurate model for plant disease detection is necessary to help farmers prevent crop loss.

## 🚀 Solution Approach
This project leverages a Convolutional Neural Network (CNN) architecture trained on thousands of crop images. The solution provides a scalable, easy-to-use user interface where farmers or agricultural experts can upload images of diseased leaves and receive instant predictions.

## 🛠️ Tech Stack
- **Languages:** Python
- **Deep Learning:** TensorFlow, Keras
- **Computer Vision:** OpenCV
- **Interface/Deployment:** Streamlit
- **Data Engineering:** NumPy, Pandas

## ✨ Features
- Deep Learning accuracy up to 95%+
- End-to-end Machine Learning pipeline (preprocessing, modeling, deployment)
- Scalable UI built with Streamlit
- Low-latency real-time inference on images

## ⚙️ How to Run
```bash
git clone https://github.com/sunnyprajapati45/Plant-Disease-Prediction-AI.git
cd Plant-Disease-Prediction-AI
pip install -r requirements.txt
streamlit run app.py
```
